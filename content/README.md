# content/

MDX content, compiled by [Content Collections](https://www.content-collections.dev)
per the config in `content-collections.ts` at the repo root.

- `docs/` — generic MDX collection (`title`, `description`, `date`, `draft`,
  `tags` frontmatter) proving the MDX pipeline compiles end-to-end.

Real content types (posts, notes, projects, etc.) and their frontmatter
contracts are an information-architecture decision made in the
product-specification phase — this collection is intentionally minimal.
